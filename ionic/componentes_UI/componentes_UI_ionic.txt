IMPORTANTE
Los componentes se nombran separando sus palabras por un guión, salvo con React,
en el cual se nombran iniciando cad palabra con mayúscula.

Ejemplo

Angular, javascript, Vue
<ion-header></ion-header>

React
<IonHeader></IonHeader>

En este documento, se nombrarán los componentes como si fueran de React
